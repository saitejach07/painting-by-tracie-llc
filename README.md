# Painting by Tracie LLC

This is a Next.js website project for Painting by Tracie LLC.

## Requirements

Before running the project, make sure you have these installed on your laptop:

- Node.js 20 or newer
- npm

## Clone The Project

```bash
git clone https://github.com/YOUR-USERNAME/painting-by-tracie-llc.git
cd painting-by-tracie-llc
```

## Install Dependencies

```bash
npm install
```

## Run The Project Locally

```bash
npm run dev
```

Then open:

```text
http://localhost:3000
```

## Create A Production Build

```bash
npm run build
npm start
```

## Project Notes

- The project uses Next.js with the App Router.
- Fonts are bundled locally so the project can run without fetching Google Fonts during build time.
